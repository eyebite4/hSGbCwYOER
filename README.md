# Python智慧社区平台简介

## 核心功能
- 车位租买
- 车位与社区信息查阅
- 周边服务
- 问卷调查
- 爱心助老
- 通知公告
- 留言反馈
- 个人中心
- 车位信息管理
- 客户信息管理
- 社区信息管理

## 适用场景
- 社区管理
- 居民生活服务
- 物业管理

## 技术特点
- 使用 Django 框架进行后端开发
- 数据库采用 MySQL
- 前端界面使用 Vue.js
- 提供源码、1万字论文及演示视频
- 支持远程调试
